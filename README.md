# lean4.kak

This is a brand new project and features are subject to change

## Dependencies

* `kakoune-lsp` - development version https://github.com/kakoune-lsp/kakoune-lsp
* `python3`

## Features
* infoview
* latex input (on InsertLeave)
* syntax highlighting

## Install
Simply source `lean4.kak` in your kakrc and keep both `lean4-replace-abbreviations.py` and `abbreviations.json` in your config directory.
